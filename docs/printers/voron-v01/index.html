Voron V0.1

The Voron V0.1 is a mini Core XY printer designed by Voron. The v0.1 can be bought in kits from multiple manufacturers, or selfsourced by following the official sourcing guide.

  • BIGTREETECH SKR 2

  • BIGTREETECH Octopus v1.1 F407
  • BIGTREETECH Octopus Pro 446
  • BIGTREETECH Octopus Pro 429
  • BIGTREETECH Octopus Pro H723
  • BIGTREETECH Manta M8P
  • BIGTREETECH SKR Pro v1.2
  • Fysetc Spider v1.1
